# Agent M - Setup Checklist

Use this checklist to track your setup progress. Check off items as you complete them.

## Prerequisites

- [ ] Node.js 18+ installed
- [ ] Python 3.11+ installed
- [ ] Git installed
- [ ] Text editor (VS Code recommended)
- [ ] Terminal/Command Prompt access

## Step 1: Clone & Install

- [ ] Cloned repository: `git clone https://github.com/ak-1344/AgentM.git`
- [ ] Ran setup script: `./setup.sh`
- [ ] Backend dependencies installed (pip)
- [ ] Frontend dependencies installed (npm)
- [ ] Virtual environment created for Python

## Step 2: Supabase Setup

- [ ] Created Supabase account at https://supabase.com
- [ ] Created new project in Supabase
- [ ] Waited for project initialization (~2 minutes)
- [ ] Ran `database/schema_phase1.sql` in SQL Editor
- [ ] Created `resumes` storage bucket
- [ ] Made bucket public for authenticated users
- [ ] Copied Project URL from Settings â†’ API
- [ ] Copied `anon` public key
- [ ] Copied `service_role` secret key
- [ ] Copied JWT Secret from JWT Settings

## Step 3: API Keys

- [ ] Created OpenAI account at https://platform.openai.com
- [ ] Generated OpenAI API key
- [ ] Saved API key securely (starts with `sk-`)

## Step 4: Backend Configuration

- [ ] Opened `backend/.env` file
- [ ] Added SUPABASE_URL
- [ ] Added SUPABASE_SERVICE_ROLE_KEY
- [ ] Added SUPABASE_JWT_SECRET
- [ ] Added OPENAI_API_KEY
- [ ] Verified ENCRYPTION_KEY is present (generated by setup.sh)
- [ ] Verified SECRET_KEY is present
- [ ] Saved the file

## Step 5: Frontend Configuration

- [ ] Opened `frontend/.env.local` file
- [ ] Added NEXT_PUBLIC_SUPABASE_URL
- [ ] Added NEXT_PUBLIC_SUPABASE_ANON_KEY
- [ ] Verified NEXT_PUBLIC_API_URL is `http://localhost:8000`
- [ ] Saved the file

## Step 6: First Run

- [ ] Opened two terminal windows
- [ ] **Terminal 1:** Started backend
  - [ ] `cd backend`
  - [ ] `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
  - [ ] `uvicorn main:app --reload`
  - [ ] Saw "Uvicorn running on http://127.0.0.1:8000"
- [ ] **Terminal 2:** Started frontend
  - [ ] `cd frontend`
  - [ ] `npm run dev`
  - [ ] Saw "ready - started server on 0.0.0.0:3000"
