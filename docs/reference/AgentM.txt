1. USER WORKFLOW (End-to-End)

Step 1: User logs in → lands on dashboard

User signs up/login (OAuth, email login, or Google Sign-In).

Sees dashboard with options:

Upload resume

Upload pitch deck / portfolio

Select context (Job Hunting / Sponsorship / Freelancing outreach etc.)

Step 2: User sets basic profile info

User provides:

Name

Email address (sender)

Phone number

Domain/context they want to target (e.g., Web dev jobs, robotics startups, sponsors for event)

Custom message / tone preference

Step 3: User provides SMTP app password

For Gmail/Outlook/Yahoo

Stored securely (encrypted, accessible only for sending)

Step 4: Agent M has a chat interaction with the user

AI asks for clarification:

Which fields to target?

What type of companies? Startup? Enterprise?

Any restrictions?

Preferred geography?

Any achievements to highlight?

User answers → A refined context document is created.

Step 5: User clicks “Start Outreach Scan”

Agent M performs:

1. Web crawling to find:

Company websites

Contact emails

Career pages

LinkedIn profiles

Startup lists

2. Filters companies by domain

3. Extracts emails / contact addresses / HR listings

Step 6: AI generates personalized emails for each target

For each company:

Reads the company website

Understands what they do

Matches with user’s resume

Generates:

Subject

Body

Custom reasoning + personalization

Forms a draft email object

Step 7: Drafts appear in “Outbound Inbox”

Here user can:

Review

Chat with AI to improve

Reject

Edit manually

Approve to send

Step 8: Upon approval → Email is sent via SMTP

Email goes directly through user’s SMTP

System logs success/failure

Step 9: Follow-up automation

(Optional)

AI can schedule follow-ups

Sends reminders

Checks replies (IMAP reading)

3. TECHNICAL WORKFLOW (DETAILED)

Phase 1 — Upload & Context Preparation

1. User uploads resume

2. Backend stores file in:

Firebase storage / Supabase storage

3. AI extracts:

Skills

Job titles

Experience

Relevant achievements

Phase 2 — AI Context Refinement Chat

1. Frontend sends user answers

2. AI creates a context.json containing:

Target roles

Preferred industries

Pitch tone

Keywords

Phase 3 — Company Discovery

Three methods:

1. Google Custom Search API

2. LinkedIn Scraping (with Playwright)

3. Startup databases like Crunchbase (free tier limited)

Crawler extracts:

Name

Website

Email IDs (regex scanning)

“Career”, “Jobs”, “Contact” pages

Phase 4 — AI Email Generation

For each company:

Fetch website text

Run AI analysis:

Company mission

Product offering

Industry

Match with resume context

AI generates personalized:

Subject

Email body

Attachment references

Draft saved to database

Phase 5 — Approval Workflow

Two modes:

Web dashboard inbox

Telegram bot approval flow

“Approve / Reject / Edit” button

If approved → send via SMTP

Phase 6 — SMTP Email Sending

Using Python smtplib or NodeMailer:

Connect to Gmail/Outlook SMTP

Use app password

Send personalized email

Phase 7 — Tracking & Logs

Store:

Delivered / Failed

Email body

Time

Follow-up scheduled
